Clinical Cases

症例集1〜20

この20数年間にレンで治療を受けていただいた方々の病気は50種類を超えます。
多くの臨床例の中から代表的なものをご紹介します。

Cases 1〜10

症例 1〜10

症例1 耳鳴り・便秘・膝の痛み

女性61歳 / 耳鍼週1回・光線週3〜5回・自宅照射毎日

治療開始4か月半で耳鳴りと便秘が解消。膝の痛みも改善。ゴルフを生きがいとされる方で、現在も継続治療中。

症例2 頭痛

女性49歳 / 耳鍼週1回

長期にわたる原因不明の頭痛と吐き気。1か月目から頭痛が楽になり、4か月で吐き気・頭痛ともに消失。1年後の来院時も「この1年間再発なし」とのうれしい言葉をいただく。

症例3 骨粗しょう症・腰の痛み

女性78歳 / 耳鍼週1回・光線週2回・自宅照射毎日

薬の副作用による骨粗しょう症。3か月頃から朝の起き上がり時の痛みが改善。5か月で日常生活の痛みが消失。約1年後の病院検査で骨密度の向上が確認され、担当医も驚く。

症例4 正座不可能・高血圧・肝機能障害

男性63歳 / 光線週6回→週2回

「もう二度と正座は出来ない」と言われたが、光線7回で正座が可能に。血圧180以上→120〜130で安定（降圧剤中止）。GOT 118→36、GPT 115→31、γ-GPT 97→54、コレステロール 273→230に改善。

症例5 顔面神経麻痺

女性31歳 / 耳鍼週1回

顔面左側の麻痺で笑うことが出来ない状態。発症から1週間以内の早期治療が功を奏し、3回の治療で解消。

症例6 顔面神経麻痺

男性53歳 / 耳鍼週1回・光線週1回

顔面右側の麻痺で病院に半年通うも効果なし。発症からの期間が長いため時間を要したが5か月で改善。

症例8 花粉症・眼精疲労

女性54歳 / 光線週2回

重度の眼精疲労と花粉症。初回治療直後から目の改善を実感。3か月でほぼ完治。花粉症も薬が不要に。光線器を購入し毎日自宅照射を継続。

症例9 視力低下

男児10歳 / 耳鍼週1回・光線週2回

治療前 右0.3・左0.6。2か月半で左右とも1.0に回復。その後1か月半で左右とも1.5に回復。

症例10 血糖値が高い・靭帯損傷

女性64歳 / 耳鍼週1回・光線週2回

血糖値311→99、GPT 46→17、中性脂肪 190→98に改善。治療途中に靭帯損傷するも、ギプスを断り光線治療で1か月で完治。

Cases 11〜20

症例 11〜20

症例11 円形脱毛症

女性35歳 / 耳鍼週1回・光線週2回・自宅照射毎日

500円硬貨大の脱毛が3か所。治療1か月目から発毛が確認され、3か月で完治。

症例12 アトピー性皮膚炎

女性22歳 / 耳鍼週1回・光線週10回（集中）・自宅照射毎日

3歳からステロイド使用。集中治療で1か月から明らかな皮膚改善。4か月でかゆみと炎症が消失。

症例13 アトピー性皮膚炎

女性30歳 / 耳鍼週1回・光線週2〜6回・自宅照射毎日

ステロイド長期使用。顔・もも・膝裏が重症。仕事を辞めて集中治療開始。4か月後には健常者の皮膚と見分けがつかないほど改善。その後エステを楽しんでいただく。

症例14 アトピー性皮膚炎

女性28歳 / 耳鍼週1回・光線週2回

比較的軽度のアトピー。ステロイドをあまり使用していないケース。遠方から週末に通院し約2年弱で完治。

症例15 耳鳴り

女性47歳 / 耳鍼週1回・光線週1回

左側耳鳴りで病院に行くも変化なし。2か月で3分の1まで改善、さらに3か月後に完治。

症例16 坐骨神経痛

女性61歳 / 耳鍼週1回・光線週1回

長年の腰痛から左側の慢性的なしびれ。2か月後からしびれのない日が出始め、6か月で完全解消。腰痛も同時に解消。

症例17 五十肩

女性51歳 / 耳鍼週1回・光線週1回

左肩拳上困難。初回の治療で腕が大幅に上がるように。2か月で完治。

症例18 五十肩

女性54歳 / 耳鍼週1回・光線週1回

右腕拳上困難と重度の肩こり。初回から大幅改善。3か月で完治。肩こりもほぼ解消。

症例19 突発性難聴

女性65歳 / 耳鍼週1回・光線週1回

突然右耳が聞こえなくなり耳鼻科に半年通うも変化なし。2か月から改善が現れ、8か月で完全回復。

症例20 難聴

女性60歳 / 耳鍼週1回・光線週1回・自宅照射毎日

5年来の難聴で右耳が全く聞こえず補聴器を使用。3か月から変化が現れ自宅照射を3倍に増量。6か月後には補聴器不要になり日常生活に支障なし。
